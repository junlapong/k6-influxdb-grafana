# k6-influxdb-grafana

https://github.com/grafana/k6

## Installation

### k6

- https://k6.io/docs/getting-started/installation/

__Build from source__

```
go install go.k6.io/k6@latest
```

### influxdb

```
apt install influxdb
```

### grafana

```
apt install grafana
```

## Running k6

```
k6 run script.js
```
